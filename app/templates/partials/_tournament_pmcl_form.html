Create Tournament Comms

PMCL (Fortunazo) tournament promo. Paste the full sheet content below — every Notification / Pop-up (Cat-fish) / SMS / Email copy field (EN/ES) comes from the paste. Works exactly like GOW Comms, but wired to the Smartico tournament deeplink instead of a promo page.

Date

Comms runs this day, 12:00 → 19:00 Chile.

Tournament ID (optional)

Smartico tournament id for the deeplink (#_smartico_dp=dp:gf_tournaments&id=…). Leave blank to keep the template's.

Journey name (optional)

Defaults to the template name (with any “Copy of ” prefix stripped).

Media Library Folder ID (optional)

When set, the script pops file pickers to upload a fresh NC icon + Pop-up background into this PMCL folder (open Media Library in the PMCL backoffice → copy the UUID from /media-library/folders/<uuid>). Leave blank to keep the template's existing images.

Spec (sheet content)

Paste the Communication channels section: Notification, Notification pop-up (Cat-fish), Email and Sms rows with their EN/ES copy. When the paste includes an Email (Title + Pre-header + Description), the email content is created + published and the journey's email activity is repointed to it; otherwise the email is left untouched.

No token needed — the script grabs it from the PMCL backoffice tab on paste, then creates the tournament comms journey draft (Notification + Pop-up + SMS).

How it works

This clones the captured PMCL tournament comms journey and swaps in the copy from your pasted sheet: the Notification (contract 1), the Pop-up (Cat-fish, contract 5), the SMS and, when present, the Email — EN & ES — all pointed at the Smartico tournament deeplink.

The send window is fixed to the chosen day, 12:00 → 19:00 Chile. The SMS carries the required Fortunazo | prefix.

If a required Notification/Pop-up/SMS field is missing from the spec, the script stops with a clear error — nothing is created.
