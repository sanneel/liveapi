How it works

Create Campaign builds the journey draft (free-spin offer with deposit tiers) and the promo page, with the campaign photo uploaded into every visual slot.

Create Communication builds the Notification Center push, the Pop-up (Cat-Fish), and the SMS — all pointed at the promo page. Tick both to create them together in one paste, with the comms links wired to the promo page created in the same run.

If a required field is missing from the spec, or the game/provider doesn't match a catalog entry, the script stops with a clear error — nothing is created.
